# Dosya Deposu

Basit dosya paylaşım sistemi. GitHub reposundaki dosyalar otomatik olarak listelenir.

## Özellikler

- ✅ Temiz liste görünümü
- ✅ Otomatik dosya algılama
- ✅ Tek tıkla indirme
- ✅ Mobil uyumlu

## Kurulum

1. GitHub'a yükle:
```bash
git push origin main
```

2. Vercel'e deploy et:
   - vercel.com'a gidin
   - Repoyu import edin
   - Deploy edin

GitHub'a yeni dosya eklediğinizde otomatik görünür.
